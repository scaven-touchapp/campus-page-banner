(File manifest) Mobile Apps' Install Banners Test Page


This page will show Appstore banner on iOS devices and mini-info bar on Android device (only in Chrome).

Sources on github

To test the android banner it is recommended to force the banner to appear by setting the chrome://flags/#bypass-app-banner-engagement-checks flag. Clear browsing data to see banner again after discarding it.

The related android app install banner will prompt the user to add your web app to the users device. It will only prompt when a number of criteria have been met:

  • The site is using HTTPS
  • The site has a manifest declared
  • The manifest has a short_name, 144 pixel icon and a type of 'image/png'
  • The manifest has a related_applications section with an object that has the 'type': 'play' and a Google Play Store package ID.
